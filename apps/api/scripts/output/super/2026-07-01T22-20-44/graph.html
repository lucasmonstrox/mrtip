Explicação

Clique num nó ou edge pra ver o texto completo do cruzamento. • Colunas da esquerda pra direita: conclusões (o que cada bloco do briefing diz) → arestas (os cruzamentos entre conclusões) → roteiros → refutação → aposta. • Passe o mouse num edge pra ver a explicação resumida; clique pra ver completa aqui. • Edges verdes = evidência a favor da aposta; vermelhos tracejados = contra / ataque da refutação; roxos = conclusão alimentando um cruzamento; teal = aresta sustentando um roteiro. • Conclusões apagadas = descartadas pelo modelo (com motivo — clique pra ver).
